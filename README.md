# 🧮 C Calculator Program

## 📌 Description

This is a simple calculator program written in C. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## 🚀 Features

* Addition
* Subtraction
* Multiplication
* Division
* User-friendly menu-driven interface

## 🛠️ Technologies Used

* C Programming Language

## 📂 How to Run

1. Compile the program using a C compiler (like GCC)
2. Run the executable file
3. Follow the menu instructions

* User input handling
* Conditional statements
* Basic program structure
